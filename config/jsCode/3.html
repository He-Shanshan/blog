# 需要理解

接口（interfaces）、泛型（generics）、类（classses）、枚举类型（enums）等

# 安装 npm i -g typescript

tsc XXX.ts react 是 .tsx
通过：指定标量的类型
编译即使出错了也会生成对应的编译结果
如果编译报错的时候终止js 文件的生成，要在tsconfig.json中配置noEmitOnError相关详细配置参考官方文档 链接

# 原始数据类型

js的数据类型分为两种：原始数据类型（primitive data types）和对象类型（object types）
原始数据类型包括：布尔类型（boolean）,数值（number）,字符串（string）,null, undefined以及es6中的新类型symbol

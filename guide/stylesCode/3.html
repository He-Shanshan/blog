# sass上手

# sass与scss简介

sass (Syntactically Awesome Stylesheets)是一种动态样式语言，Sass语法属于缩排语法，比css多出好些功能(如变量、嵌套、运算,混入(Mixin)、继承、颜色处理，函数等)，更容易阅读。

scss sass的缩排语法，对于写惯css前端的web开发者来说很不直观，也不能将css代码加入到sass里面，因此sass语法进行了改良，sass 3就变成了scss(sassy css)。与原来的语法兼容，只是用{}取代了原来的缩进。

# scss语法关键点

普通用法：
1.变量（Variables），注释（Comment），导入（Import）
2.作用域（Scope）
3.混合（Mixins）
4.嵌套（Nesting）
5.继承（Inheritance）
6.运算符（Operations）
7.颜色函数
高级应用：
1.自定义函数（function）
2.关键字参数与可变参数
3.@for简单介绍
4.@at-root 简单介绍

    margin:0 auto;
    $name:$primary-color;
}
.ass#{$name}{

    margin:0 auto;
}

# 普通用法—混合

混合就是将一部分样式抽出，作为一个单独定义的模块，被很多选择器重复使用。
使用@mixin 名称（参数）
调用的时候使用@include

@mixin error($borderWidth:2px){
    border:$borderWidth solid #f00;
    color:#f00;
}
// 调用error mixin
.generic-error{
    @include error();
}
.login-error{
    @include error(5px);
}

# 普通用法—运算符

对数值型的value（如：数字，颜色，变量等）进行加减乘除模运算，sass运算过程中会保留单位，/三种情况下会被解释为除法
1.如果值或其任何部分存储在变量中或者由函数返回
2.如果该值由括号括起，除非这些括号在列表之外并且值在内部
3.如果该值用作另一个算术表达式的一部分

p{
    font:10px/8px; //纯css，没有使用除法
    $width:1000px;
    width:$width/2;//使用变量，使用除法
    width:round(1.5)/2;//使用方法，使用除法
    height:(500px/1);//使用括号，使用除法
    margin-left:5px+8px/2px;//使用+，使用除法
    font:(italic bold 10px/8px);//在一个列表里面，括号没有生效
}

1. -作为标识符的一部分。如p-1。唯一的例外是单位; Sass通常允许将任何有效标识符用作标识符，但标识符可能不包含连字符后跟数字。这样5px-3px相同5px - 3px。
-在不带空格两个数字之间。这表示减法，因此1-2也是相同的1 - 2。
-在字面数字的开头。这表示负数，因此1 -2是包含1和-2。
-在无论有没有空格的两个数字之间。这表示减法，因此1 -$var相同1 - $var。
-在值之前。这表示一元否定; 也就是说，取一个数字并返回其负数的运算符。

# 普通用法—颜色函数

saturation($color) : 颜色的饱和度
lightness($color) : 颜色的亮度
lighten($color, $amount) :颜色更亮.
darken($color, $amount) : 颜色更暗.
saturate($color, $amount) :颜色饱和度增加.
desaturate($color, $amount) :颜色饱和度降低.
grayscale($color) :颜色的灰度.
complement($color) :颜色的补色
invert($color, [$weight]) :颜色的相反色.
mix($color, $color,$amount) :混合两种颜色

# scss语法关键点—高级应用

创建自定义函数，需要两个指令：
@function 创建函数
@return 表明了函数的返回值

@function function-name($args){
    @return value-to-be-return;
}

该指令有两种形式： @for $var from <start> through <end>和@for $var from <start> to <end>。
关键字through和to的区别，to不包括<end>值。$var可以是任何变量名称，如$i; 当<start>大于<end>计数器时将递减而不是递增。

$total:8;
@function column-width($col){
    @return percentage($col/$total);
}
@for $i from 1 through $total{
    .col-#{$i}{
        width:column-width($i);
    }
}

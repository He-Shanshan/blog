# 自定义组件

自定义组件的使用，主要是因为：
1.便于在不同页面重复使用。
2.将复杂的页面拆分成多个低耦合的模块，便于代码维护。

# 创建自定义组件

点击微信开发者工具上的+号，选择Component
类似于页面，一个自定义组件由json，wxml，wxss，js4个文件组成，要编写一个自定义组件，首先需要在json文件中进行自定义组件声明（将component字段设为true）

# 使用自定义组件

使用已注册的自定义组件前，首先要在页面的json文件中进行引用声明，此时需要提供每个自定义组件的标签名和对应的自定义组件文件路径：

{
  "usingComponents": {
    "component-tag-name":"path/to/the/custom/component"
  }
}

这样，在页面的wxml中就可以像使用基础组件一样使用自定义组件，节点名即自定义组件的标签名，节点属性即传递给组件的属性名。
tips：
1.自定义组件可以引用自定义组件，引用方法类似页面引用自定义组件的方式。
2.使用usingComponents页面的圆形与不使用时不一致，即Object.getPrototypeOf(this)结果不一致。
3.使用usingComponents时会多一些方法，如selectComponent。
4.处于性能考虑，使用usingComponents时，setData内容不会被直接深复制，即this.setData({field:obj})后this.data.field === obj.(深复制会在这个值被组件间传递时发生)

# 节点

在组件的wxml中可以包含slot节点，用于承载组件使用者提供的wxml结构。也可以简单的认为十个插槽，将节点内的结构插入到组件相应位置。

# 自定义组件例子

组件tabs

<!--components/tabs/tabs.wxml-->
<!-- 标题部分 -->
<scroll-view class="scroll-view-x" srcoll-x="true">
  <view class="scroll-view-item" wx:for="{{tabList}}" wx:key="name">
    <view class="{{currentTab==index?'on':''}}" bindtap="_switchTab" data-current="{{index}}">{{item.name}}</view>
  </view>
</scroll-view>
<!-- 内容 插槽 -->
<slot/>

// component/tabs/tabs.js
Component({
  /**
   * 组件的属性列表
   */
  properties: {
    //标题列表
    tabList:{
      type:Array,
      value:[]
    },
    //当前tab下标
    currentTab:{
      type:Number,
      value:0,
      observer:function(newVal,oldVal,data){
        //该方法当此数据被setData的时候调用
        console.error(data)
      }
    }
  },

  /**
   * 组件的方法列表
   */
  methods: {
    // 内部方法建议以下滑线开头
    _switchTab:function(data){
      this.triggerEvent("changetab",{
        currentNum:data.currentTarget.dataset.current
      })
    }
  }
})

<!--pages/component/component.wxml-->
<tabs tabList="{{statusType}}" currentTab="{{currentTab}}" bindchangetab="switchTab">
  <swiper current="{{currentTab}}" duration="300" bindchange="bindchange" style="height:{{windowHeight-35}}px">
    <swiper-item wx:for='{{list}}' wx:key='*this'>
      <view>{{item}}</view>
    </swiper-item>
  </swiper>
</tabs>

// pages/component/component.js
Page({

  /**
   * 页面的初始数据
   */
  data: {
    statusType: [
      {
      name: "待付款",
      page: 0
      },
      {
        name: "待发货",
        page: 1
      },
      {
        name: "待收货",
        page: 2
      },
      {
        name: "待评价",
        page: 3
      }, 
      {
        name: "已完成",
        page: 4
      }],
    currentTab:0,
    list: ['page1', 'page2', 'page3', 'page4', 'page5'],
    windowHeight:''
  },

  /**
   * 生命周期函数--监听页面加载
   */
  onLoad: function (options) {
    this.setData({
      windowHeight: wx.getSystemInfoSync().windowHeight
    })
  },
  switchTab:function(e){
    console.error("page",e)
    if(this.data.currentTab === e.detail.currentNum) return
    this.setData({
      currentTab:e.detail.currentNum
    })
  },
  bindchange:function(e){
    this.setData({
      currentTab:e.detail.current
    })
  }
})

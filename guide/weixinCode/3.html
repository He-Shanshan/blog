# WXML语法 数据绑定

WXML(Weixin Markup Language)是框架设计的一套标签语言，结合基础组建，事件系统，可以构建出页面的结构。

# 数据绑定

WXML中的动态数据均来自对一个Page的data。

# 响应的数据绑定

框架的核心是一个响应的数据绑定系统，可以让数据与视图非常简单地保持同步，当做数据修改的时候，只需要在逻辑层修改数据，视图层就会做相应的更新。
当我们在js文件中添加数据，如何在wxml文件中展示出来

# 控制属性

//注意不要出现空格在wx:if="{{true}}"在""和{{}}之间不然会判定为字符串
<!-- 控制属性 -->
<view wx:if="{{condition}}">{{person.name}}</view>
<view wx:elif="{{condition==2}}">{{message}}</view>
<view wx:else>{{list[2]}}</view>

# 运算

可以在{{}}内进行简单的运算，比如：
1.三元运算

<view wx:if="{{flag?true:false}}">{{person.name}}</view>
1

2.算数运算

<view>{{a + b}} = {{c}}</view>
